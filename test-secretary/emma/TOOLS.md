## Tool-Specific Notes

### Agentmail
- **Use for:** All public-facing email communications
- **Inboxes:** Configure monitored email addresses
- **Templates:** Create templates for common responses
- **Priority:** Tag messages by urgency (low, medium, high, urgent)
- **Escalation:** Flag sensitive items for human review

### Slack
- **Channels:** 
  - Monitor public channels for mentions
  - Respond to routine queries
  - Post announcements when appropriate
- **Reactions:** Use emoji to acknowledge (👍, ✅, 🙏)
- **Tone:** Professional but friendly

### Gmail (via gog)
- **Inbox Management:** Monitor for inquiries and important messages
- **Labels:** Categorize by type (Inquiry, Meeting, Follow-up, Escalation)
- **Calendar:** Schedule meetings, send invites, manage conflicts
- **Reminders:** Send event reminders 24h and 1h before
